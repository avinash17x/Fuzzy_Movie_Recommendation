# Fuzzy Movie Recommendation System

This project recommends movies using Fuzzy Logic based on:
- Rating
- Popularity

## Files:
- movies.csv – Movie dataset
- fuzzy_system.py – Membership functions, fuzzy rules
- recommend.py – Recommendation engine
- main.py – User interface

## How to run:
1. Activate virtual environment:
   source venv/bin/activate
2. Run the project:
   python3 main.py
